Python Tutorial

Learn Python - A powerful and beginner-friendly programming language

What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum in 1991. It is known for its simple syntax, readability, and versatility. Python can be used for web development, data science, automation, AI, machine learning, and more.

Why Learn Python?

  • Beginner-friendly and easy to read and write.
  • Widely used in software development, data science, AI, and automation.
  • Large community support and many libraries for almost anything.
  • High demand in the job market.
  • Portable and runs on multiple platforms (Windows, Mac, Linux).

Python Basics

Python programs are simple and readable. Some key points:

  • Case-sensitive language.
  • No need to declare data types (dynamic typing).
  • Indentation is important for defining blocks instead of braces.

Variables and Data Types

Variables are used to store data. Common data types:

  • int: Integer numbers (e.g., 10, -5)
  • float: Decimal numbers (e.g., 3.14, -0.5)
  • str: Strings (e.g., "Hello")
  • bool: Boolean values (True or False)
  • list: Collection of items (e.g., [1, 2, 3])
  • tuple: Immutable collection (e.g., (1, 2, 3))
  • dict: Key-value pairs (e.g., {"name":"Alice"})

Operators in Python

  • Arithmetic Operators: +, -, *, /, %, **, //
  • Comparison Operators: ==, !=, >, <, >=, <=
  • Logical Operators: and, or, not
  • Assignment Operators: =, +=, -=, *=, /=, %=
