C Language Tutorial

Learn C Language - The foundation of programming

What is C Language?

C is a powerful general-purpose programming language developed by Dennis Ritchie at Bell Labs in 1972. It is widely used for system programming, developing operating systems, compilers, and embedded systems.

Why Learn C Language?

  • It provides a foundation for learning other programming languages like C++, Java, Python.
  • Used in system programming and embedded systems.
  • Helps in understanding how computer works internally (memory management, pointers).
  • It is fast and efficient.
  • High demand in software development and competitive programming.

Basic Structure of a C Program

#include <stdio.h>

int main() {
    // Your code goes here
    printf("Hello World!\\n");
    return 0;
}

Explanation:

  • #include <stdio.h> - Preprocessor directive to include standard input/output library.
  • int main() - Main function where program execution starts.
  • printf() - Function used to print output on the screen.
  • return 0; - Ends the program and returns 0 to the operating system.

Data Types in C

Data types define the type of data a variable can store.

  • int: Stores integer numbers (e.g., 10, -5)
  • float: Stores decimal numbers (e.g., 3.14, -0.5)
  • double: Stores larger decimal numbers with more precision
  • char: Stores a single character (e.g., 'A', 'z')

Variables in C

Variables are used to store data. Syntax to declare a variable:

data_type variable_name;

Example:

int age;
float salary;
char grade;

Operators in C

C provides different types of operators:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=, %=

Control Statements

Control statements allow you to control the flow of execution.

  • If-Else: Conditional execution
  • Switch: Multi-way selection
  • Loops: Repeat a block of code
    • for loop
    • while loop
    • do-while loop

Functions in C

Functions are blocks of code that perform specific tasks. Syntax:

return_type function_name(parameters) {
    // code
}

Example:

int add(int a, int b) {
    return a + b;
}

Example C Program

#include <stdio.h>

int main() {
    int num1, num2, sum;
    num1 = 10;
    num2 = 20;
    sum = num1 + num2;
    printf("Sum = %d\\n", sum);
    return 0;
}

Tips to Learn C Language

  • Start with basics: variables, data types, operators.
  • Practice writing simple programs daily.
  • Understand loops, functions, and conditional statements.
  • Learn arrays, pointers, and structures gradually.
  • Try small projects like calculator, number games, or student management system.
