Java Tutorial

Learn Java - Object-Oriented Programming Language

What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995. It is platform-independent and widely used for web applications, desktop applications, mobile apps, and enterprise software.

Why Learn Java?

  • Object-Oriented Programming (OOP) language.
  • Write Once, Run Anywhere (platform-independent).
  • Used for web, mobile, and enterprise applications.
  • Strong community support and extensive libraries.
  • High demand in software development jobs.

Java Basics

Key points about Java syntax:

  • Case-sensitive language.
  • Each statement ends with a semicolon (;).
  • Program execution starts from the main method.
  • Supports classes, objects, inheritance, and polymorphism.

Example Java Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

Data Types in Java

  • Primitive Data Types: int, float, double, char, boolean, byte, short, long
  • Non-Primitive Data Types: String, Arrays, Classes, Interfaces

Variables in Java

Variables store data and must have a type. Example:

int age = 20;
float salary = 50000.50f;
char grade = 'A';
boolean isStudent = true;
String name = "Krish";

Operators in Java

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=, /=, %=

Control Statements

  • if, if-else, switch
  • for loop, while loop, do-while loop
  • break, continue, return

Functions (Methods) in Java

Methods are blocks of code performing specific tasks. Syntax:

returnType methodName(parameters) {
    // code
    return value;
}

Example:

public int add(int a, int b) {
    return a + b;
}

Tips to Learn Java

  • Start with basics: syntax, variables, data types, operators.
  • Learn OOP concepts: classes, objects, inheritance, polymorphism.
  • Practice small programs: calculator, student management, simple games.
  • Explore Java libraries and frameworks (like JavaFX, Spring) later.
  • Solve coding challenges regularly to improve problem-solving skills.
