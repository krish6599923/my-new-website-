CSS Tutorial

Learn CSS - Style your web pages

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout web pages. With CSS, you can change colors, fonts, spacing, and position elements on the page.

Why Learn CSS?

  • Enhances the appearance of web pages.
  • Controls layout, design, and responsiveness.
  • Works together with HTML and JavaScript to build complete web projects.
  • Helps create professional and modern websites.

Ways to Apply CSS

  • Inline CSS: Using style attribute inside HTML tags.
  • Internal CSS: Using <style> inside <head>.
  • External CSS: Linking a separate CSS file using <link>.

Common CSS Properties

  • Color: color, background-color
  • Text: font-size, font-family, text-align
  • Spacing: margin, padding
  • Border & Radius: border, border-radius
  • Display & Position: display, position, float

Example of CSS

<!DOCTYPE html>
<html>
<head>
    <title>CSS Example</title>
    <style>
        body {
            background-color: lightblue;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: darkblue;
            text-align: center;
        }
        p {
            color: black;
            font-size: 16px;
        }
    </style>
</head>
<body>
    <h1>Hello CSS!</h1>
    <p>This is a styled paragraph using CSS.</p>
</body>
</html>

Tips to Learn CSS

  • Start with colors, fonts, and basic text styling.
  • Practice layouts using margins, padding, and display properties.
  • Learn to style lists, links, tables, and forms.
  • Explore advanced topics like flexbox, grid, and responsive design later.
