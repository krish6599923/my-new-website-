HTML Tutorial

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML provides the basic structure of websites and is used to define elements such as headings, paragraphs, lists, links, images, tables, and forms.

Why Learn HTML?

  • It is the backbone of every website.
  • Essential for front-end web development.
  • Helps in creating structured and readable web content.
  • Allows you to integrate CSS and JavaScript to make interactive and styled websites.
  • Every web developer must know HTML to work on websites and web applications.

Basic Structure of an HTML Page

Every HTML page has a basic structure:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    Page content goes here.
</body>
</html>

Explanation:

  • <!DOCTYPE html> - Defines the document type and version of HTML.
  • <html> - The root element that contains the entire HTML document.
  • <head> - Contains metadata, title, links, and scripts.
  • <title> - Specifies the title shown in the browser tab.
  • <body> - Contains the visible content of the web page.

Common HTML Elements

Some frequently used HTML elements include:

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="url">Link Text</a>
  • Images: <img src="image.jpg" alt="description">
  • Lists: Ordered (<ol>) and Unordered (<ul>)
  • Tables: <table>, <tr>, <td>
  • Forms: <form>, <input>, <textarea>, <button>

Example of a Simple HTML Page

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first web page using HTML.</p>
    <a href="https://www.example.com">Visit Example</a>
</body>
</html>

Tips to Learn HTML Effectively

  • Start by understanding the basic tags and their purpose.
  • Practice creating small web pages daily.
  • Learn how to structure content properly using headings, paragraphs, and lists.
  • Experiment with links, images, and tables.
  • Once comfortable, move to CSS and JavaScript for styling and interactivity.
